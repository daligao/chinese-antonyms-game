# Chinese Antonyms Matching Game — 反义词连连看

A browser-based Chinese antonyms matching game (连连看) with pinyin and English hints.

**[▶ Play the full version (380 pairs) →](https://ordinarymantrying.com/tools/chinese-antonyms.html)**

## Demo

This repo contains a **2-round demo** (24 antonym pairs across 3 difficulty levels).

Open `index.html` in any browser — no server, no build step, no dependencies.

## Features

- 3 difficulty levels: Easy (single characters) / Medium / Hard (compound words)
- Pinyin + English hints on every tile
- Combo multiplier scoring system
- Hint button, skip round, timer
- Mobile-friendly (2-column grid on small screens)
- Pure HTML/CSS/JS — zero dependencies

## Full Version

The full game has **380 antonym pairs** (133 easy + 123 medium + 124 hard) sourced from a curated Chinese antonym dictionary, with pinyin and English for every pair.

👉 **[ordinarymantrying.com/tools/chinese-antonyms.html](https://ordinarymantrying.com/tools/chinese-antonyms.html)**

## Screenshot

The game shows 8 tiles per round (4 antonym pairs). Click a character, then click its opposite to match. Correct pairs turn green; wrong pairs shake red.
